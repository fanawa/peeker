# peeker
- Web予約時の診察券番号や、交通事故時の緊急連絡など、必要な情報を画像とともに登録しておけばスムースに確認できます。
- アプリ内データベースを使用するので、情報が外部漏洩するリスクが低いです。

# 機能
- ホーム画面 リスト/アイコン表示に切り替え可能
- 詳細画面 登録した電話番号への発信、URLをブラウザ表示
